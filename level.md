# XP Grinding Guide 

* [Meta Gear for Leveling](https://forums.wynncraft.com/threads/meta-gear-for-leveling.275840/)


## Level 1-10

* Literally just walk around in circles and stab a few mobs in the back and you are level 10

## Level 10-29

* Use 2 morph topaz + durums serenity + master
* Mt. Wynn with a healer, skill everything into defense
* Use the first xp potions

## Level 29-36

* Mt Wynn, use full adventurer set 

## Level 36-50

* Herb
* use adventurer set
* use durums serenity


## Level 50-65 

* Use [Illumite, any chestplate that gives xp, Chained Pixels, Seven League Boots] to solo herb, use adventurer if you are with a healer

## Level 65-69

* Use Bad wolf + the set named above

## Level 69-84

* Use Blue Mask, Jingler Jester, Kelights Shield, Bad Wolf + [Criistal, 2 Morph Topaz, Vanguard]

## Level 84-100

* Use Bantisus Approach, Speaker, Ensas Ideals, Dragon Hide Plate + rings above
* Do the highest possible dungeon
